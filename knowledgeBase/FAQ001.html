How to cutomise the batch file settings?

To cutomise the batch file settings, please follow the step below:

  1. Go to the GameObject VGear, VGear (Script) > [System] > Batches .

  2. Each element of the list stored will be generated with the corresponding parameters.

  3. Modify the existing element or add a new element by changing the Size of the list.

  4. For each element, there are Type, Name, Config Name, Reference Config and Setting Name can be modified.

    Note: Config Name, Reference Config and Setting Name are the path to the configuration files. The path will be appended to the corresponding configuration folder if the path stored is not started with Assets. For example, the setting file read will be Built application\VotanicXR\Settings.

  5. Modify the Type with different environment types if needed, the Name and Config Name will be changed automatically.

    Name will be concatenated to the application name into the batch file generated. The default value will be the chosen environment type warpped by bracket.

    Config Name stores the path of the configuration file of the application. The default value will be the corresponding config of the chosen environment type.

  6. Modify the Reference Config and Setting Name if needed.

    Reference Config will create transparent screens for the reference in the application, the result will be similar to the transparent screen created by the reference config in the editor. The default value of the Reference Config will be null.

    Setting Name stores the path of the setting file of the application. The default value will be setting.vxrs.

    Note: Config Name, Reference Config and Setting Name

  7. After build the project, the batch files will be built with customized settings.
