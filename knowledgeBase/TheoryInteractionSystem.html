Interaction System

To interact between interactor and interactables, the interactable object should have Interactables component. The default interactors are called Wand and Vision.

vCast Interactables

The basic interactables is called vCast interactables. You can set grabbable and interaction setting to make the game object be grabbable and triggerable. About the interaction setting, you can learn more in API reference document.

You have at least 3 ways to add the component:

  1. Menu Bar Component > VotanicXR > vCast > Interactables
  2. Right Click the game object, select vCast > Component > Interactables
  3. Click the game object Add Component button, find VotanicXR > vCast > Interactables, or just search

vGear Interactables

The advanced type of interactables for vGear only, also called interactable+. We recommend you use vGear interactables instead of vCast interactables. You can set more than one interaction. You can add this component using the similar way to vCast interactables.

vGear Button

The animatable version of vGear interactables that can be added to 3D object or canvas UI element. You can set the transition effect of this interactables.

vGear Panel

The animatable interactables that used to group a set of buttons.

vGear Slider

The slidable version of vGear button that can slide by triggering. You can set the range of value and default value of the slider. The change of value and transform is controlled by the controller transform change. You can also set a scroll content to the slider. The content will scroll when the value of slider is changing.
