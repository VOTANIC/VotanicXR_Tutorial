XR Configurations

VotanicXR configuration system files are the important files for different equipment and different projects to have a correct performance.

Config files

The system environment configuration file. Projects on an equipment can share the same config file in global directory. Config file includes Node (Entity, Camera and Screen configs) and Projection (including Multi-display and Console Mode) config which are related to the XR platform.

Setting files

The project setting file. Each project may have different setting file. Setting file includes Scene (Mode, Update, Quality, Frame and User settings), Device and Command setting which are related to the project.

Option files

The project optional setting file. This file is only available for the project using vGear. As the settings on this file are optional, developer should think what elements the project needed. Option file includes Built-in Menu, Landing Menu and Teleport setting.
