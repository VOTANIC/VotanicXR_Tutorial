Software Requirement

The following software are required for using VotanicXR and running VotanicXR applications:

Operating System

VotanicXR SDK for Unity and the compiled VotanicXR applications works best with Windows 10 (64-bit), though it is also compatible with PCs running Windows 7 SP3 or above.

Unity3D

VotanicXR SDK for Unity is compatible with Unity2018.2 to Unity 2019.4 for all supported XR platforms, and may work with Unity 2020.1 and later versions of Unity for the non-HMD playback.

Shared Code Library

VotanicXR SDK for Unity and the compiled VotanicXR applications require some shared code libraries to function correctly. The following code libraries are required:

  • Microsoft Visual C++ Redistributable 2012
  • Microsoft Visual C++ Redistributable 2013
  • Microsoft Visual C++ Redistributable 2019

These shared code libraries will be installed automatically during the installation of VotanicXR Suite.

VotanicXR Player License

The VotanicXR Player License is required to play VotanicXR applications in CAVE or multi-display systems. Please request a license from us here.

XR System Drivers

The following XR system drivers are required for using VotanicXR and running VotanicXR applications:

SteamVR

Steam VR software from Steam is needed if you wish to use the OpenVR Plugin or build applications for any Head-mounted Display (HMD) system. Download and install the Steam VR software at

https://store.steampowered.com/app/250820/SteamVR/

Oculus Software

The Oculus software for any Oculus HMD is needed if you wish to use an Oculus HMD device. Download and install the software at https://www.oculus.com/setup/

Windows Mixed Reality for SteamVR

SteamVR and Windows Mixed Reality plugin on SteamVR is needed if you wish to use Windows Mixed Reality device. Download andinstall the software at

https://store.steampowered.com/app/719950/Windows_Mixed_Reality_for_SteamVR/

VotanicXR is compatible with any passive stereoscopic system.

VR Headsets

  • SteamVR compatible VR headsets
  • Oculus headsets
  • Windows Mixed Reality VR headsets
