Adding Media

MediaPlayer

VGear have provided MediaPlayer feature with Virtual Reality environment, will be introduced below.

  1. Right-click vGear > GameObject > MediaPlayer.

  2. Go to the component Media Player (Script) of the child, select the Media in [Settings].

  3. For Size, set the desired number of the media want to play. For each element below, set the media file path to Path.

    You can also set the whole directory to Path to play all the media in the folder and set the Type to filter the undesired media type.

    For Path, we suggested to use external resources folder with path ../Resources/* and place this media folder next to the build.

The media file supported for the MediaPlayer is shown below:

Audio: .wav, .ogg,
Image: .jpg, .png, .tga, .exr,
Video: .mp4, .mov, .avi, .mkv, .asf, .webm

For each type of media you may set the setting such as image dutation or audio channel.

The below will introduce some of the remaining settings.

[Settings]
Ignore List Path of text file of the media list wanted to be ignored in current launch.
The list should includes the file name of the ignored media with the format.
Auto Play Enable to auto play media.
Auto Next Enable to auto open next media.
Random Enable to randomly play media.
Loop Loop setting of media player, All to loop all media, One to loop one medium.
[Player UI] The settings to store the UI components of the Player.
[Status] The settings to store and display the status of the player, can be get/set by provided functions.

MediaPlayer360

VGear have provided MediaPlayer360 for the 360 degree video and Images.

To add the prefab, right-click vGear > GameObject > MediaPlayer360.

Most components of the MediaPlayer360 is the same as MediaPlayer, settings of [360] about the 360 media will be introduced below.

[360]
Data Path Path of the json file which store the media data in the path.
Recommented to use external resources folder with path "../Resources/*".
Zoomer Transform for zoom.
Rotater Transform for rotate.
Pitcher Transform for pitch angle.
Calibrator Calibrate factor for calibrator.
Logo Element Logo Element for hiding top and bottom of 360 medium.
Zoom Speed Zoom speed of zoomer.
Rotate Speed Rotate speed of rotater.
