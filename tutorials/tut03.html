Teleport Tool

Teleport is common navigation tool in VR application, VotanicXR have provided it with editable effects.

Using Default Teleport Tool

Set Teleport as Default Tool

  1. Go to vGear > Frame > Controller > Teleport, component V Gear_Teleport (Script) is shown at the inspector window, which manage the setting of the Teleport tool.

    And go to vGear > Frame > Controller to set the default tool into Teleport for testing.

  2. Enter play mode, the default tool is changed into teleport.

    By pressing the Trigger button of the controller, or send the Teleport command through Input System, the vGear user will be teleported into the projection point.

    {
	//	Press Space to Teleport
	if (vGear.Input.KeyboardPress(KeyCode.Space))
	vGear.Cmd.Send("Teleport");
}

Teleport using Waypoints

To make sure player teleport into a correct position, Waypoint can be used.

  1. Right click vGear > GameObjects > Waypoint and drag the prefab into desired position.

    To apply a custom rotation, you can enable the Apply Rotation with custom value of face angle and offset in V Gear_Waypoint (Script).

Using Controller

  1. Connected a controller accpeted by VotanicXR and open the scene created at previous chapter.

  2. Click play to enter the play mode. Game window will be shown with the perspective view of the vGear user at the start point with the default wand tool.

  3. Use directional pad to navigate forward, backward and rotate.

Using Inut System

Also, send the Move command and Rotate command with specified value through Input System, can navigate forward, backward and rotate.

{
//  Using Keyboard WASD through DirectInput to move
    if (vGear.Input.KeyboardPress(KeyCode.W))
        vGear.Cmd.Send("Move", 1);
    if (vGear.Input.KeyboardPress(KeyCode.S))
        vGear.Cmd.Send("Move", -1);
    if (vGear.Input.KeyboardPress(KeyCode.D))
        vGear.Cmd.Send("Rotate", 1);
    if (vGear.Input.KeyboardPress(KeyCode.A))
        vGear.Cmd.Send("Rotate", -1);
}

To use the teleport function in different ways, we can set different modes in setting [Advance] of component V Gear_Teleport (Script).

[Advanced]
Teleport Waypoint Waypoint detection setting.
Detect will detect waypoint, Ignore will ignore waypoint,
and WaypointOnly only detect waypoint for teleportation.
Collision Mode Enable to detect collisions to prevent teleportation,
else only teleport to the ground level.
